#### 🔨 압축 명령어
소스반영시 기존 소스를 백업하기 위해 해당 명령어 사용
```shell
# tar 로 압축하기
tar -cvf [압축파일명.tar] [압축할 대상]

tar -cvf ROOT.tar ~/ROOT/* # ROOT 하위 모든 파일 압축

# tar.gz 로 압축하기
tar -zcvf [압축파일명.tar.gz] [압축할 대상]
```
